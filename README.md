# Singular TimeZones

## Dependencies

- Nodejs
- Apache Cordova
- Android SDK
- IOS SDK
- Bower

## Use
### Global installs
```
$ npm install -g gulp
$ npm install -g bower
$ npm install -g cordova
```
### Local installs
```
$ npm install
$ bower install
```
### Development command
```
$ gulp
```
### Production commands
```
$ gulp build
$ gulp server-www

$ cordova build [platform]
$ cordova emulate [platform]
```
